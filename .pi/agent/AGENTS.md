# Agent Instructions

## Commit Messages

- Always write commit messages according to the Conventional Commits specification.

## Neovim Configuration

- My Neovim configuration is located at `~/.config/nvim`.
- I use [LazyVim](https://github.com/lazyvim/lazyvim) as my Neovim distribution.
- All plugins are installed under `~/.local/share/nvim/lazy/` including LazyVim itself.
- Inspect plugin source code when you need to understand internals, defaults, plugin specs, or behavior.
